## 主要文件说明

### 核心组件

#### App.vue
根组件，只包含 `<router-view></router-view>` 用于渲染当前路由组件。

#### Layout.vue
主布局组件，包含：
- 侧边栏菜单（文章管理、用户管理）
- 用户信息显示和操作（头像、退出登录）
- 路由出口显示子页面

### 路由配置 (router/index.js)

定义了应用的路由嵌套结构：
- `/login`：登录页面
- `/`：主布局页面，包含子路由：
  - `/article/category`：文章分类管理
  - `/article/manage`：文章管理
  - `/user/info`：用户信息
  - `/user/avatar`：用户头像
  - `/user/resetPassword`：重置密码
---
### 状态管理 (stores/)

#### token.js
管理用户认证 Token：
- `token`：存储 JWT Token
- `setToken()`：设置 Token
- `removeToken()`：清除 Token
- 配置持久化存储

#### userInfo.js
管理用户信息：
- `info`：存储用户详细信息
- `setInfo()`：设置用户信息
- `removeInfo()`：清除用户信息
- 配置持久化存储
---
### API 接口封装

#### utils/request.js
Axios 实例配置：
- 基础 URL 设置为 `/api`（通过 Vite 代理到后端）
- 请求拦截器：自动添加 Authorization 头
- 响应拦截器：统一处理业务状态码和错误

#### api/user.js
用户相关 API：
- `userRegisterService()`：用户注册
- `userLoginService()`：用户登录
- `userInfoService()`：获取用户信息
- `userInfoUpdateService()`：更新用户信息
- `userAvatarUpdateService()`：更新用户头像

#### api/article.js
文章相关 API：
- `articleCategoryListService()`：获取文章分类列表
- `articleCategoryAddService()`：添加文章分类
- `articleCategoryUpdateService()`：更新文章分类
- `articleCategoryDeleteService()`：删除文章分类
- `articleListService()`：获取文章列表
- `articleAddService()`：添加文章
---
### 页面组件

#### Login.vue
登录/注册页面：
- 支持用户名密码登录
- 支持用户注册
- 表单验证（用户名 5-16 字符，密码 5-16 字符）
- 调用登录/注册 API

#### 文章管理页面
- **ArticleCategory.vue**：文章分类管理（增删改查）
- **ArticleManage.vue**：文章内容管理

#### 用户管理页面
- **UserInfo.vue**：查看和编辑用户信息
- **UserAvatar.vue**：上传和更换用户头像
- **UserResetPassword.vue**：修改用户密码
--- 
## 与后端连接

### 代理配置
在 `vite.config.js` 中配置开发服务器代理：
```javascript
server: {
  proxy: {
    '/api': {
      target: 'http://localhost:8080',
      changeOrigin: true,
      rewrite: (path) => path.replace(/^\/api/, '')
    }
  }
}
```

### 请求流程
1. 前端发起请求时，使用 `/api` 前缀
2. Vite 开发服务器将 `/api` 请求代理到 `http://localhost:8080`
3. 后端接收到请求（路径已重写，去掉 `/api`）
4. 响应返回给前端

### 认证机制
- 登录成功后，后端返回 JWT Token
- 前端将 Token 存储在 Pinia store 中（持久化）
- 后续请求通过请求拦截器自动添加 `Authorization` 头
- 响应拦截器统一处理认证失败的情况

## 技术栈

- **前端框架**：Vue 3 (Composition API)
- **构建工具**：Vite
- **路由管理**：Vue Router 4
- **状态管理**：Pinia (带持久化插件)
- **UI 组件库**：Element Plus
- **HTTP 客户端**：Axios
- **样式预处理器**：Sass
- **富文本编辑器**：Vue Quill

## 运行项目

1. **安装依赖**：
   ```bash
   npm install
   ```

2. **启动开发服务器**：
   ```bash
   npm run dev
   ```

3. **构建生产版本**：
   ```bash
   npm run build
   ```

4. **预览生产构建**：
   ```bash
   npm run preview
   ```
