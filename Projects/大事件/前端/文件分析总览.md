```javascript

```
3. **退出登录功能**：

### ArticleManage.vue - 文章管理

- **文件位置**：`src/views/article/ArticleManage.vue`
- **功能**：管理文章的列表展示、搜索、添加功能。

1. **数据模型**：
```javascript
const categorys = ref([])    // 分类列表
const articles = ref([])     // 文章列表
const categoryId = ref('')   // 搜索条件：分类ID
const state = ref('')        // 搜索条件：发布状态
const pageNum = ref(1)       // 当前页
const total = ref(20)        // 总条数
const pageSize = ref(3)      // 每页条数
```

2. **分页处理**：
```javascript
const onSizeChange = (size) => {
    pageSize.value = size
    articleList()  // 重新获取数据
}
const onCurrentChange = (num) => {
    pageNum.value = num
    articleList()
}
```

3. **获取文章列表**：
```javascript
const articleList = async () => {
    let params = {
        pageNum: pageNum.value,
        pageSize: pageSize.value,
        categoryId: categoryId.value ? categoryId.value : null,
        state: state.value ? state.value : null
    }
    let result = await articleListService(params);
    total.value = result.data.total;
    articles.value = result.data.items;
    
    // 处理数据：添加分类名称
    for (let article of articles.value) {
        for (let category of categorys.value) {
            if (article.categoryId == category.id) {
                article.categoryName = category.categoryName;
            }
        }
    }
}
```

4. **添加文章**：
```javascript
const articleModel = ref({
    title: '',
    categoryId: '',
    coverImg: '',
    content: '',
    state: ''
})

const addArticle = async (clickState) => {
    articleModel.value.state = clickState;  // '已发布' 或 '草稿'
    let result = await articleAddService(articleModel.value);
    ElMessage.success(result.msg ? result.msg : '添加成功');
    visibleDrawer.value = false;  // 关闭抽屉
    articleList()  // 刷新列表
}
```

5. **文件上传**：
```javascript
const uploadSuccess = (result) => {
    articleModel.value.coverImg = result.data;  // 保存封面图片URL
}
```

**模板结构**：
- 搜索表单（分类选择、状态选择、搜索按钮）
- 文章列表表格（标题、分类、时间、状态、操作）
- 分页组件
- 添加文章抽屉（包含富文本编辑器 Vue Quill）
